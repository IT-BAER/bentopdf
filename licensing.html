PDF Tools Lizenz

PDF Tools ist Open-Source-Software und wird unter der MIT-Lizenz veröffentlicht. Dies bedeutet, dass Sie die Software frei nutzen, modifizieren und verteilen können.

MIT-Lizenz erlaubt:

  • Kommerzielle Nutzung
  • Modifikation des Quellcodes
  • Verteilung und Weitergabe
  • Private Nutzung

Verwendete Bibliotheken

PDF Tools nutzt verschiedene Open-Source-Bibliotheken. Wir sind den Entwicklern dieser Projekte dankbar für ihre hervorragende Arbeit.

pdf-lib

MIT

Bibliothek zum Erstellen und Bearbeiten von PDF-Dokumenten in JavaScript.

PDF.js (Mozilla)

Apache 2.0

PDF-Rendering-Engine von Mozilla für die Anzeige von PDFs im Browser.

Coherent PDF

LGPL

Leistungsstarke PDF-Bearbeitungsbibliothek für erweiterte Operationen.

Tailwind CSS

MIT

Utility-First CSS-Framework für das moderne Design.

Font Awesome

Font Awesome Free

Icon-Bibliothek für die Benutzeroberfläche.

Danksagungen

PDF Tools wäre ohne die großartige Open-Source-Community nicht möglich. Wir danken allen Entwicklern und Mitwirkenden der verwendeten Bibliotheken sowie allen, die zu diesem Projekt beitragen.
